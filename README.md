# HSMI Intelligence Stack

AI operations stack for **Hepburn Springs Motor Inn**.

## Modules

| Folder | Purpose |
|--------|---------|
| `pricing_engine/` | Dynamic room pricing based on occupancy, demand, and local events |
| `chat_agent/` | Guest-facing chat assistant for enquiries, bookings, and FAQs |
| `voice_agent/` | Phone/voice assistant for inbound calls |
| `post_stay/` | Post-stay follow-up automation (reviews, feedback, re-engagement) |
| `shared/` | Shared utilities, API clients, and configuration |

## Setup

1. Copy `shared/.env.example` to `shared/.env` and fill in your credentials.
2. See each module's README for individual setup instructions.
